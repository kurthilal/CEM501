Classifier

Returns URGENT / ACTION / FYI / ARCHIVE. Uses Anthropic when ANTHROPIC_API_KEY is set in project/.env; otherwise keyword rules from demo.reader.triage_email.

Drafter

Generates a plain-text reply (Claude if API key present; else templates).

Full pipeline

One click: classify → draft (same as scheduler logic without persisting here).

Inbox preview

Calls demo.reader.read_recent_emails (IMAP). Requires valid EMAIL_* in project/.env.

Telegram

Run python3 -m demo.test_bot from project/ to receive messages. Reply from here when WEB_ALLOW_SEND=1 and TELEGRAM_BOT_TOKEN are set in project/.env.

Active chats
Select a chat to view messages and photos.

Memory (SQLite)

Reads demo/memory/memory.db. Load messages loads email/agent rows and Telegram rows in two separate tables.

SMTP send (optional)

Disabled by default. Set WEB_ALLOW_SEND=1 in project/.env to enable — only use on a trusted machine.

Scheduler & Telegram (CLI)

These are not started by the web server — run from project/:

cd project
python3 -m demo.scheduler --once --dry-run --max-emails 5
python3 -m demo.scheduler --dry-run --interval-mins 10
python3 -m demo.test_bot
